## 程序员的自我修养


- 作者: 俞甲子 / 石凡 / 潘爱民
- 出版社: 电子工业出版社
- 出品方: 博文视点
- 副标题: 链接、装载与库
- 出版年: 2009-4
- 页数: 459
- 定价: 65.00
- 装帧: 平装16开
- ISBN: 9787121085116

*这本书主要介绍系统软件的运行机制和原理，涉及在Windows和Linux两个系统平台上，一个应用程序在编译、链接和运行时刻所发生的各种事项，包括：代码指令是如何保存的，库文件如何与应用程序代码静态链接，应用程序如何被装载到内存中并开始运行，动态链接如何实现，C/C++运行库的工作原理，以及操作系统提供的系统服务是如何被调用的。*
